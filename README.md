# MyReads App

Una aplicación de gestión de libros construida con React y Vite.

## 🚀 Cómo empezar

- Clone o descargue el proyecto
- Ejecute `npm install` en el directorio raíz del proyecto para instalar las dependencias
- Ejecute `npm run dev` para lanzar el proyecto

## ✨ Características

- Gestión de libros en diferentes estanterías (Leyendo, Quiero leer, Ya leídos)
- Búsqueda de libros con debounce para optimizar rendimiento
- Validación de props con PropTypes
- Interfaz responsiva con CSS moderno
